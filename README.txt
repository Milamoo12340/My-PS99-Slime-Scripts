=== PS99 SLIME TYCOON DOMINATOR ===
Version 5.3.2
Custom version for Milamoo12340 (Jacquie)

=== INSTALLATION ===
1. Make sure AutoHotkey is installed on your computer
2. Run the "Run_SlimeTycoon_Dominator.bat" file to start the script

=== HOW TO USE ===
1. Launch the script and the control panel will appear
2. Join the Slime Tycoon area in Pet Simulator 99
3. Press F8 to select your Roblox window
4. Press F1 while hovering over the center of the area you want to farm
5. Press F6 or click the START DOMINATOR button
6. The script will automatically detect and break slime blobs for maximum points

=== GLOBAL HOTKEYS ===
F1: Set target area center
F2: Trigger Burst Mode
F3: Trigger Instant Break
F4: Trigger Pulse Wave 
F5: Force scan for slimes
F6: START DOMINATOR
F7: STOP DOMINATOR
F8: Select Roblox Window
F9: Emergency stop

=== EXECUTION MODES ===
- Balanced: Moderate speed and power with good combo building
- Speed: Maximum speed but less precision
- Power: Focuses on high damage per break
- Stealth: Minimizes detection risk with humanized movements
- Extreme: Maximum performance with all features enabled
